## Mise en page des documents (Word, PDF, PowerPoint)

### En-tête
- **Barre Orange Auguria `#FFBA00`** sur toute la largeur en haut de page en dessous de l'en-tête (épaisseur ~4pt)
- **Logo Auguria** : petit format, aligné à gauche
- **Titre du document** : centré ou à droite du logo, en Bleu Auguria `#185ADB`
- **Date** : alignée à droite
- **Mention de confidentialité** : "CONFIDENTIEL" ou "Document confidentiel — Auguria" visible dans l'en-tête

### Pied de page
- **Barre Orange Auguria `#FFBA00`** sur toute la largeur en bas de page au dessus du pied de page (épaisseur ~4pt)
- **Mention de confidentialité** : répétée en pied de page (ex. "Document confidentiel — Diffusion restreinte")
- **Numéro de page** : aligné à droite (ex. "Page X / Y")
- **Adresse Paris** : alignée à gauche — `Auguria — 10, rue de la Paix — 75002 Paris`

### Styles de texte (Word / DOCX)
- Titres H1 : Bleu Auguria `#185ADB`, gras
- Titres H2 : Bleu foncé `#0D2760`
- Accents / encadrés : Orange `#FFBA00`
- Corps de texte : Gris foncé `#333333`

### PowerPoint / PPTX
- Fond des slides : blanc `#FFFFFF`
- Barre Orange `#FFBA00` en haut et bas de chaque slide
- Éléments d'accent : Orange `#FFBA00`
- Texte principal : Gris foncé `#333333`
- Logo en bas à droite ou en en-tête à gauche
- Mention de confidentialité en pied de slide

### Excel / XLSX
- En-têtes de colonnes : fond Bleu Auguria `#185ADB`, texte blanc
- Lignes d'accent / totaux : Orange `#FFBA00`
- Texte : Gris foncé `#333333`

## Module Odoo — Page descriptive (`static/description/index.html`)

### Structure générale
La page `index.html` de chaque module Odoo suit un gabarit **cartouche industriel** conforme à l'identité Auguria. Elle est affichée sur l'Odoo Apps Store et dans le back-office Odoo (menu Apps). Le design s'inspire du style visible sur les pages apps tierces Odoo (ex. `apps.odoo.com`).

### Cartouche en-tête
- **Bordure** : cadre `3px solid #185ADB`, border-radius `4px`, ombre légère `rgba(13, 39, 96, 0.10)`
- **Barres orange** : bande `#FFBA00` de `6px` en haut et en bas du cartouche
- **3 colonnes** disposées en `display: flex` :
  1. **Logo** : logo Auguria complet (bleu/orange) intégré en base64, centré, `max-height: 52px`, séparé par un trait bleu `2px solid #185ADB`
  2. **Titre du module** : surtitre "MODULE ODOO" en `#FFBA00` (10px, uppercase, letter-spacing 2px), nom du module en `#185ADB` (22px, bold), sous-titre descriptif en `#333333` (12px)
  3. **Infos techniques** : fond `#F5F5F5`, labels en `#0D2760` (bold), valeurs en `#333333` — Version, Licence, Auteur, Catégorie

### Sections de contenu (Description, Fonctionnalités)
- **Titres H2** : `#185ADB`, 18px, bold, avec bordure gauche `4px solid #FFBA00` et padding-left `12px`
- **Corps de texte** : `#333333`, 13px, line-height `1.7`, padding-left `16px`
- **Listes** : `#333333`, 13px, line-height `2`, padding-left `32px`

### Onglets (Dépendances / Technique)
Système d'onglets **CSS pur** (sans JavaScript), compatible avec le rendu Odoo Apps Store.

#### Mécanisme
- Utilisation de `<input type="radio">` cachés + sélecteurs CSS `:checked ~`
- Classe conteneur : `.aug-tabs`
- Labels dans `.aug-tab-labels`, panneaux dans `.aug-tab-panel`

#### Styles des onglets
- **Onglet inactif** : fond `#F5F5F5`, texte `#0D2760`, border-radius `6px 6px 0 0`
- **Onglet actif** : fond `#185ADB`, texte `#FFFFFF`
- **Barre sous les onglets** : `border-bottom: 3px solid #185ADB`
- **Panneau** : `border: 2px solid #185ADB`, border-top none, border-radius `0 0 6px 6px`, fond blanc

#### Onglet 1 — Dépendances
- **Sous-titre** : classe `.aug-tech-title` — `#0D2760`, 14px bold, `border-bottom: 2px solid #FFBA00`
- **Badges modules** : classe `.aug-badge` — fond `#185ADB`, texte blanc, 12px bold, padding `4px 14px`, border-radius `3px`
- Sections : "Modules Odoo requis" + "Modules optionnels"

#### Onglet 2 — Technique
Trois sous-sections avec tableaux :

**Vues impactées** — Tableau `.aug-tech-table` :
- En-têtes : fond `#185ADB`, texte blanc, uppercase, 11px
- Colonnes : Vue (XML ID en `<code>`), Type, Modèle
- Lignes paires : fond `#F9FAFE`, hover `#EEF2FB`
- Code inline : fond `#F0F3FA`, texte `#0D2760`

**Groupes de sécurité & Droits** — Matrice CRUD :
- Colonnes : Groupe, XML ID, Lecture, Écriture, Création, Suppression
- Badge "Utilisateur" : `.aug-badge-outline` (bordure `#185ADB`, fond blanc)
- Badge "Manager" : `.aug-badge` (fond `#185ADB`, texte blanc)
- ✓ autorisé : `#27AE60` bold / ✗ refusé : `#E74C3C` bold, centré

**Modèles de données** — Tableau :
- Colonnes : Modèle, Description, Type
- Badge "Persistant" : `.aug-badge-outline`
- Badge "Transient" : fond `#FFF3CD`, texte `#856404`, bordure `#FFBA00`

### Pied de cartouche
- **Bordure** : cadre `2px solid #185ADB`, border-radius `4px`
- **Barres orange** : bande `#FFBA00` de `4px` en haut et en bas
- **Contenu** : fond `#F5F5F5`, flex space-between — à gauche "**Auguria** — Consultant & Intégrateur Odoo — Nantes & Paris" (`#0D2760` bold pour le nom), à droite lien `auduria.fr` en `#185ADB` bold

### Référence logo dans le HTML
- Le logo Auguria complet (bleu/orange) est intégré directement en **base64** dans la balise `<img src="data:image/png;base64,..." />`
- L'icône du module (`static/description/icon.png`) reste le **logo G condensé** orange, redimensionné en `128×128 px` (format standard Odoo)
